# Monitor de Ruido Urbano - Proyecto IoT

Este repositorio contiene el desarrollo del sistema de monitoreo de ruido ambiental utilizando una Raspberry Pi Pico W.

## Semana 1: Boceto/CAD Preliminar del Encapsulado

En esta primera semana se diseñó la estructura de la carcasa protectora en Tinkercad:
* **Base Roja:** Incluye la ranura para el cable Micro-USB de alimentación y 4 postes internos de soporte para fijar la Raspberry Pi Pico W.
* **Tapa Frontal Verde:** Cuenta con una ventana calibrada para el Semáforo LED (actuador visual) y un orificio para el Micrófono MAX4466 (sensor acústico).
